STARCRAFT UNIT API

This project provides an API endpoint for general unit data for the units within the competitive Versus mode of the Blizzard game, Starcraft II. As a side web development project, I went looking for this endpoint, and couldn't find it available online. So, I put this together to provide JSON data for the units, including wiki link, photo, resource costs, descriptions, and various other bits of data. All info was webscraped from the excellent fan made Starcraft II wiki.


There is a link to the API endpoint above, in the navbar. The API also contains endpoints for each specific race, if you do not want data from the other two races. There are also delete, update, and create routes that are tokenized. If any errors are discovered, please let me know.

API ROUTES

  • Protoss Units
  • Terran Units
